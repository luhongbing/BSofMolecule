# 分子结构编辑器

一个基于Web的有机物分子3D结构编辑器，支持实时编辑和查看分子模型，提供共线共面分析功能。

## 功能特性

### 核心功能
- **3D分子模型渲染**：使用Three.js实现流畅的3D球棍模型渲染
- **实时交互**：支持鼠标拖拽旋转、滚轮缩放、右键平移
- **分子编辑**：支持添加/删除原子和化学键
- **共线共面分析**：自动识别共线原子和共面原子，支持手动选择分析

### 工具栏功能
- 原子选择工具（C/H/O/N/卤素等常见有机元素）
- 化学键工具（单键/双键/三键）
- 视图控制（旋转、重置视角、显示/隐藏标签）
- 渲染模式切换（球棍模型、空间填充、线框）
- 预设分子模板（甲烷、乙烯、苯、乙醇等）
- SMILES格式导入

### 分析功能
- 自动计算分子式、分子量、不饱和度
- 杂化轨道分析（sp/sp2/sp3）
- 共线原子检测
- 共面原子检测
- 手动选择原子进行共线共面分析

## 技术栈

- **框架**: React 18 + TypeScript
- **3D渲染**: Three.js
- **样式**: Tailwind CSS 3
- **构建工具**: Vite

## 使用说明

### 基本操作
1. **旋转模型**: 鼠标左键拖拽
2. **缩放**: 滚轮滚动
3. **选择原子**: 点击原子（按住Shift可多选）
4. **添加原子**: 在工具栏选择原子类型，然后在画布上点击

### 快捷键
- **Shift + 点击**: 多选原子

### SMILES格式支持
支持基本的SMILES格式输入，例如：
- `C` - 甲烷
- `C=C` - 乙烯
- `C#C` - 乙炔
- `c1ccccc1` - 苯

## 核心算法

### 共线检测
使用向量叉积判断三点是否共线：如果三个点的向量叉积模长小于阈值，则判定为共线。

### 共面检测
使用平面方程判断四点是否共面：先由前三点确定平面方程，再计算第四点到平面的距离，若距离小于阈值则判定为共面。

### 杂化轨道分析
根据碳原子的键级总和判断杂化类型：
- 键级总和为4 → sp³杂化
- 键级总和为3 → sp²杂化
- 键级总和为2 → sp杂化
